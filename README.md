# Foundry Modules

Dépôt regroupant mes modules Foundry VTT personnalisés.

## Modules

- `event-forge` : Interface custom pour créer des événements avec test de compétence, fluff et log journal.
- `gaia-exploration-tools` : Outils d'exploration pour la planète Gaïa.
- `lib-foundry-core` : Librairie utilitaire agnostique système pour mutualiser les interactions communes avec Foundry VTT.
- `lib-pf2e-tools` : Librairie utilitaire pour PF2e / SF2e.
- `log-semi-auto` : Résumé semi-automatique des combats avec ajout au journal.
- `pf2e-hero-stats` : Suit les statistiques des jets de dés et assiste la gestion des points d'héroïsme pour PF2e et SF2e.
- `pf2e-spellshot-helper` : Automatisation partielle des capacités du Way of the Spellshot pour Pathfinder 2e.
- `sanguimancer-resource` : Gestion de la ressource sanguimancie.
- `vn-widget` : Ajoute un widget Vitality Network sur la fiche des personnages.
